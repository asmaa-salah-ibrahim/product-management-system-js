# Product Management System (JavaScript)

A simple product management system built with vanilla JavaScript to practice core front-end concepts.

## Features
- Add, update, and delete products
- Search products by name
- Data persistence using LocalStorage
- Input validation using Regular Expressions

## Technologies Used
- JavaScript (Vanilla JS)
- HTML5
- CSS / Bootstrap
- LocalStorage

## Purpose
This project was built to practice JavaScript fundamentals, focusing on data flow, state changes, and proper input validation rather than UI design.
